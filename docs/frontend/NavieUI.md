## Navie UI

网址：[Naive UI: 一个 Vue 3 组件库](https://www.naiveui.com/zh-CN/os-theme)

Naive UI是一个仅支持vue3的库，有超过80个组件，希望能帮你少写点代码。使用的是由TypeScript构建的先进类型安全主题系统，运行快、小巧轻量、专门针对样式优化，不需要导入任何css就能让组件正常工作。
